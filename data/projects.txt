Проект: Smart AI Agent  
Ссылка: https://github.com/maks-mk/simple-ai-agent

Описание:
Автономный AI-агент на Python с поддержкой инструментов и самокоррекции.

Используемые технологии:
- Python  
- LangGraph / LangChain  
- Google Gemini / OpenAI  
- Model Context Protocol (MCP)  
- AsyncIO  

Реализованные возможности:
- Оркестрация действий через граф состояний  
- Вызов внешних инструментов  
- Встроенный веб-поиск  
- Механизм самовосстановления при ошибках  
- Долгосрочная память  
- Стриминговый CLI-интерфейс  
- Защита от зацикливания  
- Гибкая конфигурация через .env и mcp.json
