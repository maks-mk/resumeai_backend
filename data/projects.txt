Проект: Smart AI Agent
Ссылка: https://github.com/maks-mk/simple-ai-agent
Роль: Автор и разработчик (Pet-проект)

Описание:
Автономный AI-агент на Python. Архитектура построена по принципу «LLM как мозг, инструменты как исполнители». Агент самостоятельно планирует шаги, выбирает инструменты и корректирует стратегию при ошибках.

Технологический стек:
- Python, AsyncIO
- LangGraph (реализация графа состояний с циклами reasoning -> tool -> validation)
- Google Gemini / OpenAI API
- Model Context Protocol (MCP) для интеграции инструментов

Ключевые возможности:
- Самокоррекция: механизм Retry при ошибках API или логики.
- Инструменты: веб-поиск, работа с файловой системой, запуск команд.
- Интерфейс: Стриминговый CLI-вывод (библиотека Rich).
- Безопасность: защита от зацикливания и контроль доступа к файлам.
